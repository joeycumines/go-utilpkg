# jsonenc

Package jsonenc implements JSON encoding utilities, providing zero-allocation
append-style functions for strings and numbers.

## Documentation

Available [here](https://pkg.go.dev/github.com/joeycumines/go-utilpkg/jsonenc).

## Attribution

The string encoding is derived from
[zerolog](https://github.com/rs/zerolog)'s implementation, with extensions.
See [string.go](string.go) for the full license notice.
